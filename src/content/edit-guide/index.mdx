# Editing Guide for MDX Pages

Welcome! This guide will help you edit and create content for this site using MDX (Markdown + JSX). No prior experience is required.

## What is MDX?
MDX lets you write Markdown (for text formatting) and embed React/JSX components. Itâ€™s great for content-rich pages.
<br/>
## Basic Markdown Syntax

- **Headings:**
  - `# Heading 1`
  - `## Heading 2`
  - `### Heading 3`
- **Bold:** `**bold text**`
- **Italic:** `*italic text*`
- **Lists:**
  - Unordered: `- Item 1`  
  - Ordered: `1. First item`
- **Links:** `[Link text](https://example.com)`
- **Images:** `![Alt text](path/to/image.png)`
- **Code:**
  - Inline: `` `code` ``
  - Block:
    ```
    ` ` `
    code block
    ` ` ` 
    (without the spaces)
    ```
<br/>
## Embedding Components
You can use custom components like `<InfoCard />` or `<SectionHeading />` directly in your MDX files:

```jsx
<InfoCard title="Example" description="This is an example info card." />
```
<br/>
## "Frontmatter" - your file metadata
At the top of each MDX file, you can add metadata:

```
---
title: "Page Title"
description: "Short description."
---
```
<br/>
## Tips
- Save images in the `src/assets/` folder and reference them with a relative path.
- Use components from `src/components/mdx/` for rich layouts.
- Preview your changes locally before publishing.
